--React Native--

-Create React Native App-

cd into directory you want to create app in

expo init projectname --template blank

-Open project in VS Code-
code .

-To start app development tools-
expo start
open Android Studio 
Click Configure  
Select AVD Manager
Click play 

OR 
expo start --tunnel
Scan barcode with camera or expo app
